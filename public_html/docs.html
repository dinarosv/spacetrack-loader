Documentation for satellites API

/sat

Fetch all satellites

Parameters:

  • ID: fetch item with given ID (NORAD_CAT_ID)
  • FILE: fetch data newer than given FILE no. Also returns decayed items.

When not specifying any parameters, all satellites (not decayed) are returned

How to

Example: http://folk.ntnu.no/dinaro/sat?FILE=2716026

  • ? indicates start of parameters
  • Use & between multiple parameters
